Dataset Files

hdbflat_resaleprices.txt – Text file containing the Google Drive link to the 23-year HDB resale prices dataset.

**Download CSV:**  
https://drive.google.com/drive/folders/19vQ7ulawlTkYZrAGBf9n0UwZf4AWYjSh?usp=sharing

Contains historical resale transactions including town, flat type, floor area, storey range, and transaction date.

region.csv – Mapping of towns to regions, stored locally for easy access.

Useful for regional comparison, aggregation, and visual analysis in Power BI.

# Power BI Dashboard

This folder contains the Power BI dashboard developed for the Singapore HDB Resale Price Analysis project.

The dashboard visualizes key insights derived from the HDB resale transaction dataset.

## Key Visualizations

- Resale price trends over time
- Average resale price by town and region
- Impact of flat age and flat type on resale pricing
- Top 10 town by price and transaction
- classify town

## Dashboard Features

- Interactive filters for year, town, and flat type
- Drill-down analysis for deeper exploration
- KPI indicators for market trends

The dashboard helps identify pricing patterns, high-demand towns, and market changes in Singapore’s HDB resale market.
